# Introduction

主要專注在 Leetcode 前400題的刷題攻略，扣除掉 20 題 database, shell 相關的題目，總共剩下 380 題，
分成 18 類別：Array, String, Math, Tree, Backtracking, DP, Linked List, Binary Search, Matrix, DFS & BFS, Stack & Queue, Bit Manipulation, Topological Sort, Random, Graph, Union Find, Trie, Design.

分類參考來源：
* https://vbnetvbnet.github.io/cs/#nav-jieshao
* https://www.cspiration.com/#/leetcodeClassification

其實還有很多刷題攻略，我也只是選擇其中一條路而已，賭上身家後就全力衝刺，也接受之後的結果。
